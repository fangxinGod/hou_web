缺陷检测模型优化

PyTorch CNN Transformer CLIP 多尺度学习

优化目标

在有限的标注数据下，提升现有Sewer-ML、MAQ2L等下水道缺陷检测模型的准确率（特别是召回率），并尽可能保持较高的推理速度。

核心优化策略

多尺度特征融合

将传统的单尺度卷积替换为多尺度感受野模块，使用特征金字塔FPN进行不同尺度的特征提取，增强模型对不同大小缺陷的识别能力。

视觉-语言语义增强

引入CLIP模型的先验知识，通过文本提示（如“裂缝”）增强模型对特定缺陷类别的语义理解。

轻量级记忆模块

增加一个小型记忆模块——Memory-Unit，使得模型在关联相关类别信息的同时，还会对一些视频抽取出的序列进行上下文信息的关联，这个自学习内存单元模块的设计，也为后续的模型推理更好的创造了条件。

核心修改代码

以下代码展示了多尺度特征融合 + 轻量级记忆模块的核心实现部分：


        class MultiScaleMemoryNet(nn.Module):
            def __init__(self, backbone, memory_size=128):
                super().__init__()
                self.backbone = backbone
                self.memory = nn.Parameter(torch.randn(memory_size, 256))  # Memory-Unit
                self.fpn = FeaturePyramidNetwork(
                    in_channels_list=[256, 512, 1024],
                    out_channels=256
                )

            def forward(self, x):
                feats = self.backbone(x)              # 多尺度特征提取
                feats = self.fpn(feats)               # FPN 融合
                memory_context = torch.matmul(
                    feats[0].flatten(2), self.memory.t()
                )                                     # 关联记忆模块
                return feats, memory_context
